# Provide some explanations on why you believe the performance metrics you chose is appropriate
performance_metrics_explanations = """
This is a 3-class task with imbalance (Standard 53%, Poor 29%, Good 18%). Accuracy alone would overvalue the majority class.

Macro-F1: computes F1 per class then averages. Each class counts the same. Good for imbalance.

Balanced accuracy: averages recall per class. It checks if the model misses minority classes.

Macro ROC-AUC (OvR): uses probabilities to measure overall class separability. Less sensitive to threshold choice.

The classification report and confusion matrix has been included to see which classes are confused and why. These reveal concrete error patterns for business impact.
"""
